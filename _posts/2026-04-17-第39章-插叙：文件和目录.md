> **核心问题**：如何使用文件系统接口？

## 本章大纲

### 39.1 文件和目录

**文件**：线性字节序列，持久存储

**目录**：文件和其他目录的集合（树状结构）

### 39.2 文件系统接口

```c
#include <fcntl.h>
#include <unistd.h>
```

### 39.3 创建文件

```c
int fd = open("newfile.txt", O_WRONLY | O_CREAT | O_TRUNC, 0644);
// O_CREAT: 不存在则创建
// O_TRUNC: 存在则截断
// 0644: 权限 (rw-r--r--)
```

### 39.4 读写文件

```c
// 写入
ssize_t bytes_written = write(fd, buffer, size);

// 读取
ssize_t bytes_read = read(fd, buffer, size);
```

### 39.5 读取和写入，但不按顺序

```c
// 定位到文件偏移量
off_t lseek(int fd, off_t offset, int whence);
// whence: SEEK_SET, SEEK_CUR, SEEK_END

// 读取文件偏移 100 处的数据
lseek(fd, 100, SEEK_SET);
read(fd, buf, size);
```

### 39.6 用 fsync() 立即写入

```c
write(fd, data, size);  // 数据可能在缓冲区
fsync(fd);              // 强制写入磁盘
```

### 39.7 文件重命名

```c
rename("old.txt", "new.txt");
```

### 39.8 获取文件信息

```c
struct stat sb;
stat("file.txt", &sb);

// sb.st_size: 文件大小
// sb.st_mode: 文件类型和权限
// sb.st_mtime: 修改时间
```

### 39.9 删除文件

```c
unlink("file.txt");  // 删除文件的目录项
```

### 39.10 创建目录

```c
mkdir("newdir", 0755);
```

### 39.11 读取目录

```c
DIR *dir = opendir(".");
struct dirent *entry;

while ((entry = readdir(dir)) != NULL) {
    printf("%s\n", entry->d_name);
}

closedir(dir);
```

### 39.12 删除目录

```c
rmdir("emptydir");  // 目录必须为空
```

### 39.13 硬链接

```c
link("original.txt", "link.txt");
// 两个文件名指向同一 inode
```

### 39.14 符号链接

```c
symlink("original.txt", "softlink.txt");
// 软链接指向文件名
```

### 39.15 创建并挂载文件系统

```c
// 创建文件系统
mkfs("disk.img");

// 挂载
mount("disk.img", "/mnt/point");
```

### 39.16 小结

| 操作 | 系统调用 |
|------|----------|
| 创建文件 | open() |
| 读写文件 | read(), write() |
| 定位 | lseek() |
| 同步 | fsync() |
| 重命名 | rename() |
| 删除 | unlink() |
| 目录操作 | mkdir(), rmdir(), readdir() |
