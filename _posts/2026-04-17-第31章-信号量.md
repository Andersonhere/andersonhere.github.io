> **核心问题**：信号量如何用于同步？

## 本章大纲

### 31.1 信号量的定义

**Dijkstra 信号量**：

```c
sem_t s;
sem_init(&s, 0, 1);  // 初始值 = 1

sem_wait(&s);   // P操作：减1，如果<0则阻塞
// 临界区
sem_post(&s);   // V操作：加1，唤醒等待者
```

### 31.2 二值信号量（锁）

```c
sem_t lock;
sem_init(&lock, 0, 1);  // 初始值 1

sem_wait(&lock);
// 临界区
sem_post(&lock);
```

**等价于**：
- `sem_wait` = `lock`
- `sem_post` = `unlock`

### 31.3 信号量用作条件变量

```c
sem_t cond;
sem_init(&cond, 0, 0);  // 初始值 0

// 线程 A 等待
sem_wait(&cond);  // 阻塞

// 线程 B 通知
sem_post(&cond);  // 唤醒 A
```

### 31.4 生产者/消费者问题

```c
sem_t empty, full, mutex;
sem_init(&empty, 0, MAX);  // 空槽位
sem_init(&full, 0, 0);     // 满槽位
sem_init(&mutex, 0, 1);    // 互斥

void producer() {
    while (1) {
        sem_wait(&empty);  // 等待空槽
        sem_wait(&mutex);
        put(item);
        sem_post(&mutex);
        sem_post(&full);   // 增加满槽
    }
}

void consumer() {
    while (1) {
        sem_wait(&full);   // 等待满槽
        sem_wait(&mutex);
        item = get();
        sem_post(&mutex);
        sem_post(&empty);  // 增加空槽
    }
}
```

### 31.5 读者—写者锁

**问题**：
- 多读者可同时读
- 写者需要独占访问

```c
sem_t lock, writelock;
int readers = 0;

void read_lock() {
    sem_wait(&lock);
    readers++;
    if (readers == 1)
        sem_wait(&writelock);
    sem_post(&lock);
}

void read_unlock() {
    sem_wait(&lock);
    readers--;
    if (readers == 0)
        sem_post(&writelock);
    sem_post(&lock);
}

void write_lock() {
    sem_wait(&writelock);
}

void write_unlock() {
    sem_post(&writelock);
}
```

### 31.6 哲学家就餐问题

**问题**：5个哲学家围坐，每人需要左右两根筷子

**解决方案**：

```c
sem_t forks[5];

void philosopher(int i) {
    while (1) {
        think();
        // 按顺序拿筷子
        sem_wait(&forks[i]);
        sem_wait(&forks[(i+1) % 5]);
        eat();
        sem_post(&forks[i]);
        sem_post(&forks[(i+1) % 5]);
    }
}

// 死锁解决方案：最后一个哲学家先拿右边的
```

### 31.7 如何实现信号量

```c
typedef struct {
    int value;
    pthread_mutex_t lock;
    pthread_cond_t cond;
} sem_t;

void sem_wait(sem_t *s) {
    pthread_mutex_lock(&s->lock);
    s->value--;
    if (s->value < 0) {
        pthread_cond_wait(&s->cond, &s->lock);
    }
    pthread_mutex_unlock(&s->lock);
}

void sem_post(sem_t *s) {
    pthread_mutex_lock(&s->lock);
    s->value++;
    pthread_cond_signal(&s->cond);
    pthread_mutex_unlock(&s->lock);
}
```

### 31.8 小结

**信号量用途**：

| 用途 | 初始值 | 说明 |
|------|--------|------|
| 锁 | 1 | 互斥访问 |
| 条件变量 | 0 | 线程同步 |
| 计数器 | N | 资源计数 |
