> **核心主题**：CPU 虚拟化知识总结

## 核心机制回顾

### 底层机制

| 机制 | 目的 |
|------|------|
| 用户/内核模式 | 保护系统安全 |
| 陷阱指令 | 进入内核模式 |
| 时钟中断 | 定期获取控制权 |
| 上下文切换 | 切换运行进程 |

### 关键抽象

**进程**：运行中程序的抽象
- 地址空间
- 寄存器状态
- 打开的文件
- 其他资源

### 调度策略

| 策略 | 优化目标 |
|------|----------|
| FIFO | 简单 |
| SJF/STCF | 周转时间 |
| Round Robin | 响应时间 |
| MLFQ | 综合（无需预知） |
| 比例份额 | 公平分配 |

## 关键问题与答案

| 问题 | 答案 |
|------|------|
| 如何高效执行？ | 受限直接执行 |
| 如何保持控制？ | 时钟中断 |
| 如何选择进程？ | 调度策略 |
| 如何保证公平？ | MLFQ / 比例份额 |
