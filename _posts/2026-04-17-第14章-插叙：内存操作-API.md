> **核心问题**：程序如何分配和管理内存？

## 本章大纲

### 14.1 内存类型

| 类型 | 存储位置 | 生命周期 |
|------|----------|----------|
| 栈内存 | 栈 | 函数调用期间 |
| 堆内存 | 堆 | 手动管理 |
| 静态/全局 | 数据段 | 程序运行期间 |

### 14.2 malloc() 调用

```c
#include <stdlib.h>

void *malloc(size_t size);
// 成功返回指针，失败返回 NULL

// 使用示例
int *arr = (int *)malloc(10 * sizeof(int));
if (arr == NULL) {
    // 处理错误
}
```

### 14.3 free() 调用

```c
void free(void *ptr);

// 使用示例
int *arr = malloc(10 * sizeof(int));
// ... 使用 arr
free(arr);  // 释放内存
arr = NULL; // 避免悬空指针
```

### 14.4 常见错误

| 错误类型 | 说明 |
|----------|------|
| 忘记分配 | 使用未初始化的指针 |
| 忘记释放 | 内存泄漏 |
| 重复释放 | double free |
| 释放后使用 | 悬空指针 |
| 越界访问 | 缓冲区溢出 |

```c
// 常见错误示例

// 1. 忘记分配
int *p;
*p = 10;  // 错误：未初始化

// 2. 内存泄漏
void leak() {
    int *p = malloc(sizeof(int));
    return;  // 忘记 free
}

// 3. 重复释放
int *p = malloc(sizeof(int));
free(p);
free(p);  // 错误

// 4. 释放后使用
int *p = malloc(sizeof(int));
free(p);
*p = 10;  // 错误：悬空指针
```

### 14.5 底层操作系统支持

**系统调用**：

| 调用 | 功能 |
|------|------|
| `brk/sbrk` | 调整堆大小 |
| `mmap` | 内存映射 |

### 14.6 其他调用

```c
// calloc: 分配并初始化为 0
void *calloc(size_t nmemb, size_t size);

// realloc: 调整已分配内存大小
void *realloc(void *ptr, size_t size);

// 内存对齐分配
int posix_memalign(void **memptr, size_t alignment, size_t size);
```

### 14.7 小结

**内存管理最佳实践**：
1. 检查 malloc 返回值
2. 成对使用 malloc/free
3. 释放后置 NULL
4. 使用工具检测内存问题（valgrind）
