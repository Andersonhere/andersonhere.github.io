> **核心问题**：如何使用线程 API？

## 本章大纲

### 27.1 线程创建

```c
#include <pthread.h>

int pthread_create(
    pthread_t *thread,           // 线程标识
    const pthread_attr_t *attr,  // 线程属性(可NULL)
    void *(*start_routine)(void *), // 线程函数
    void *arg                     // 传递参数
);
```

**示例**：

```c
pthread_t t1;
int rc = pthread_create(&t1, NULL, mythread, "hello");
if (rc != 0) {
    perror("pthread_create");
}
```

### 27.2 线程完成

```c
int pthread_join(pthread_t thread, void **retval);
```

**等待线程结束并获取返回值**：

```c
void *retval;
pthread_join(t1, &retval);
// retval 包含线程返回值
```

### 27.3 锁

```c
pthread_mutex_t lock;

// 初始化
pthread_mutex_init(&lock, NULL);
// 或静态初始化
pthread_mutex_t lock = PTHREAD_MUTEX_INITIALIZER;

// 使用
pthread_mutex_lock(&lock);
// 临界区
pthread_mutex_unlock(&lock);

// 销毁
pthread_mutex_destroy(&lock);
```

### 27.4 条件变量

```c
pthread_cond_t cond = PTHREAD_COND_INITIALIZER;
pthread_mutex_t lock = PTHREAD_MUTEX_INITIALIZER;

// 等待条件
pthread_mutex_lock(&lock);
while (条件不满足) {
    pthread_cond_wait(&cond, &lock);
}
// 条件满足，继续执行
pthread_mutex_unlock(&lock);

// 通知条件变化
pthread_mutex_lock(&lock);
// 改变条件
pthread_cond_signal(&cond);
pthread_mutex_unlock(&lock);
```

### 27.5 编译和运行

```bash
gcc -o myprogram myprogram.c -lpthread
./myprogram
```

### 27.6 小结

| API | 功能 |
|-----|------|
| `pthread_create` | 创建线程 |
| `pthread_join` | 等待线程结束 |
| `pthread_mutex_lock/unlock` | 加锁/解锁 |
| `pthread_cond_wait` | 等待条件 |
| `pthread_cond_signal` | 通知条件 |

## 使用模式

### 基本线程模式

```c
void *thread_func(void *arg) {
    // 线程工作
    return NULL;
}

int main() {
    pthread_t threads[N];
    
    // 创建线程
    for (int i = 0; i < N; i++) {
        pthread_create(&threads[i], NULL, thread_func, NULL);
    }
    
    // 等待所有线程
    for (int i = 0; i < N; i++) {
        pthread_join(threads[i], NULL);
    }
    
    return 0;
}
```

### 生产者-消费者模式

```c
pthread_mutex_t lock = PTHREAD_MUTEX_INITIALIZER;
pthread_cond_t cond = PTHREAD_COND_INITIALIZER;
int buffer[MAX];
int count = 0;

void put(int value) {
    pthread_mutex_lock(&lock);
    while (count == MAX) {
        pthread_cond_wait(&cond, &lock);
    }
    buffer[count++] = value;
    pthread_cond_signal(&cond);
    pthread_mutex_unlock(&lock);
}

int get() {
    pthread_mutex_lock(&lock);
    while (count == 0) {
        pthread_cond_wait(&cond, &lock);
    }
    int value = buffer[--count];
    pthread_cond_signal(&cond);
    pthread_mutex_unlock(&lock);
    return value;
}
```
