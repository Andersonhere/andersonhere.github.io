> **核心主题**：引入内存虚拟化的概念

## 本章要点

### 为什么需要内存虚拟化？

**问题**：
- 多个程序同时运行，如何隔离？
- 程序如何使用超过物理内存的空间？
- 如何让每个程序都认为自己独占内存？

### 内存虚拟化的目标

1. **透明性**：程序不需要知道内存虚拟化的存在
2. **效率**：时间和空间开销尽可能小
3. **保护**：进程之间相互隔离

### 地址空间

每个进程拥有独立的虚拟地址空间：

```
进程 A 的地址空间          进程 B 的地址空间
┌──────────────┐          ┌──────────────┐
│   栈 ↓       │          │   栈 ↓       │
├──────────────┤          ├──────────────┤
│     ...      │          │     ...      │
├──────────────┤          ├──────────────┤
│   堆 ↑       │          │   堆 ↑       │
├──────────────┤          ├──────────────┤
│   数据       │          │   数据       │
├──────────────┤          ├──────────────┤
│   代码       │          │   代码       │
└──────────────┘          └──────────────┘
      ↓                         ↓
   物理内存                  物理内存
```
