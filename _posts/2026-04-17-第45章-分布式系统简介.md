> **核心问题**：分布式系统面临哪些挑战？

## 本章要点

### 分布式系统的挑战

| 挑战 | 说明 |
|------|------|
| 通信失败 | 网络不可靠 |
| 节点故障 | 机器可能宕机 |
| 性能 | 网络延迟 |
| 一致性 | 多副本同步 |

### 通信抽象

```
客户端                    服务端
   │                        │
   │──── 请求 ────────────→│
   │                        │ 处理请求
   │←─── 响应 ─────────────│
   │                        │
```

### 核心问题

**问题1：不可靠的通信层**

```
消息可能:
- 丢失
- 延迟
- 重复
- 乱序
```

**问题2：可靠的通信层**

```
可靠传输需要:
- 确认机制
- 重传
- 序列号
- 去重
```

### 远程过程调用 (RPC)

**目标**：让远程调用看起来像本地调用

```c
// 客户端
result = remote_add(1, 2);

// 实际发生:
// 1. 序列化参数
// 2. 发送到服务器
// 3. 服务器执行
// 4. 返回结果
// 5. 反序列化结果
```

**RPC 步骤**：

```
客户端:                    服务端:
1. 调用 stub
2. 序列化参数               4. 反序列化
3. 发送请求      ────────→  5. 调用实际函数
                           6. 序列化结果
7. 反序列化    ←───────────  7. 发送响应
8. 返回结果
```
